# AI’Han Lab

## Tarayıcıda Çalışan Python, Veri Bilimi ve Yapay Zekâ Laboratuvarı

**AI’Han Lab**, Python programlamayı, veri analizini, veri görselleştirmeyi, makine öğrenmesini ve yapay zekâ uygulamalarını tarayıcı içinde deneyimleyerek öğretmek için geliştirilmiş etkileşimli bir eğitim portalıdır.

Portal yalnızca konu anlatımı veya hazır kod örnekleri sunmaz. Kullanıcılar Python kodlarını doğrudan tarayıcıda çalıştırabilir, değişkenleri değiştirebilir, grafik ve tablolar üretebilir, model sonuçlarını inceleyebilir ve kendi verileriyle uygulamalar geliştirebilir.

**Geliştiren:** Ayhan Bozkurt
**AI’Han Academy:** https://ayhanbzkrt.github.io/benaihan/

## Yerel Sunucuyla Çalıştırma

Bazı tarayıcılar yerel dosyalarda güvenlik sınırlamaları uygulayabilir. Bu durumda HTML dosyasının bulunduğu klasörde basit bir yerel sunucu başlatılabilir.



2. Portal HTML dosyasını depoya ekle.

6. Kaynak olarak ana dalı ve kök klasörü seç.

## `pip install` ve `import` Farkı

Bir paketi bilgisayara indirmek için:

```bash
pip install pandas
```

Paketi çalışan Python programına bağlamak için:

```python
import pandas as pd
```

`pip install`, paketi sisteme kurar.
`import`, kurulu paketi çalışmakta olan programa dâhil eder.

Tarayıcı sürümünde uyumlu paketler Pyodide tarafından gerektiğinde otomatik olarak yüklenir.
